# 00 Expected First Response

## Stage

State 0: Intake → State 1: Socratic Interview

## What I Understand

你要做一个科技品牌的吸尘器 VI。当前明确的信息是：行业为消费科技 / 清洁家电，产品方向是吸尘器，目标是完整 VI。

## Missing Information / Confirmed Assumptions

还不能生成 logo 图像，因为以下关键信息缺失：

- 品牌名或是否需要命名
- 具体产品类型：无线吸尘器、扫拖机器人、洗地机、除螨仪、商用设备等
- 目标用户和价格带
- Logo 灵感或允许探索的视觉路线
- 品牌气质和竞品参照
- 必须避免的风格
- 首要落地场景
- 是否确认要完整 VI 手册

## Output / Options

请先回答下面几个问题，我再进入设计方向和 logo 图像生成阶段：

1. 品牌名是否已经确定？如果没有，需要我一起提出命名方向吗？
2. 核心产品是哪一种：无线吸尘器、扫拖机器人、洗地机、除螨仪，还是全屋清洁系统？
3. 目标用户是谁：年轻家庭、宠物家庭、高端住宅用户、科技爱好者，还是商用场景？
4. 价格带希望是大众科技、精品日常、高端旗舰，还是奢华智能家居？
5. Logo 灵感更偏向什么：气流、涡轮、传感器、洁净路径、字母 monogram、产品轮廓，还是纯文字标？
6. 品牌气质希望更接近：工程精密、安静克制、未来速度、温和家居、环保健康，还是专业商用？
7. 有哪些一定要避免的风格：卡通吸尘器、廉价小家电、过度赛博、医疗蓝、家政服务感、复杂机械纹理？
8. VI 最重要的落地场景是什么：产品机身、包装、电商详情页、App、说明书、售后卡、门店展台，还是社交广告？
