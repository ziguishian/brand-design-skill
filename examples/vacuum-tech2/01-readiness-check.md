# 01 Readiness Check

## Input Brief

做一个科技品牌，吸尘器的品牌 VI。

## Readiness Checklist

| Requirement | Status | Notes |
| --- | --- | --- |
| brand/product name or naming placeholder | Missing | Brand name not provided |
| industry and product category | Partial | Technology + vacuum, but product type unspecified |
| hero product/service | Missing | Could be cordless vacuum, robot, wet/dry cleaner, etc. |
| target audience | Missing | Needed for style and tone |
| price tier or market position | Missing | Needed to avoid generic tech aesthetics |
| desired personality/tone | Missing | Needed for design direction |
| logo inspiration or allowed inspiration routes | Missing | Required before image generation |
| avoid list | Missing | Required to prevent wrong style |
| primary application scenarios | Missing | VI must match real touchpoints |
| VI scope | Partial | User asks VI, but manual depth not confirmed |

## Decision

Blocked before image generation.

The agent should ask Socratic questions and wait. It should not create logo prompts, generate images, SVG files, or HTML manuals yet.
