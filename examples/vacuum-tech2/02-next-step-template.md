# 02 Next Step Template

After the user answers the Socratic questions, the agent should:

1. Summarize confirmed details.
2. List any remaining assumptions.
3. Ask for confirmation if assumptions are still needed.
4. Create a design direction system.
5. Present 3 to 5 logo image direction options.
6. Ask the user to choose which directions to generate or say "generate all".

Only after the user says to generate should the agent enter State 3 and call image generation.

After image generation, the agent must show the images and ask the user to choose one of these actions:

1. Choose one candidate to convert into SVG.
2. Ask for a refinement round.
3. Combine specific elements from candidates.
4. Reject all and revise the design direction.
5. Ask the agent to recommend the strongest candidate.
